Git Workflow Documentation

Follow these steps to create and manage branches, commit changes, and merge them using Git:

1. Navigate to Desktop and Create a Folder

cd /c/Users/YOUR_USERNAME/Desktop
mkdir my_folder
cd my_folder

2. Initialize Git and Create a File

echo "Hello from my_folder!" > myfile.txt
git init

3. Check Current Branch and Status

git branch    # Shows current branch
git status    # Shows untracked files

4. Stage and Commit Changes

git add .
git commit -m "Initial commit with myfile.txt"

5. Create a New Branch

git branch new_branch    # Create a new branch

6. Switch to the New Branch

git checkout new_branch  # Move to the new branch

7. Make Changes and Commit on the New Branch

echo "Changes made in new_branch" >> myfile.txt
git add .
git commit -m "Updated myfile.txt in new_branch"

8. Switch Back to Main and Merge Changes

git checkout main
git merge new_branch

9. Delete the Merged Branch

git branch -d new_branch
